#LearnKotlin App 

##About the app
The app is a compilation of the apps in the Kotlin Basics Pathway (Unit 1 - 4). The different activities contain the following apps:
- Birthday Card App
- Dice Game App
- Tip Calculator App
- Affirmations App
- Word Game App
- Dessert Clicker App
- Unscramble Game App
- Mars Photos App
